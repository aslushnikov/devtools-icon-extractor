To export these icons to the file system:
  1. Create empty icons folder
  2. Open DevTools, run copy(generateCommands().join(';')).
  3. There's a bash script in the clipboard now. Save it as icons/gen.sh
  4. Run the following in your Terminal
    5. $ cd icons/
$ bash gen.sh             # script will create many SVG files
$ npm install -g svgo     # install SVG optimizer/fixer
$ mkdir -p ./optimized;   # create folder for optimized files
$ for i in *.svg; do svgo --multipass --pretty $i optimized/$i; done
